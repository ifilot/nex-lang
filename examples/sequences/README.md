# Sequence Examples

This folder contains small NEX programs built around numeric sequences. They
show how far you can get with a handful of variables, loops, and helper
functions, even without arrays.

Run any example with:

```bash
python3 -m nex.cli examples/sequences/<file>.nex
```

## Scripts

- `triangular_numbers.nex`: prints the first few triangular numbers using running accumulation.
- `powers_of_two.nex`: doubles a value repeatedly to generate powers of two.
- `fibonacci.nex`: generates Fibonacci numbers with two evolving state variables.
- `collatz.nex`: prints the Collatz sequence for a starting value until it reaches `1`.
- `factorials.nex`: prints successive factorial values using iterative multiplication.
