# Reference

This section describes the currently supported NEX language core. It is written
as a language reference, but it is also meant to be readable by students who
want to understand how the pieces of a programming language fit together.

The emphasis is on source-level behavior rather than implementation details.
That means the chapters below describe what a NEX program means, what kinds of
values it can manipulate, how scopes are formed, and what kinds of errors are
part of the language contract. Unless noted otherwise, examples in this section
are expected to work in the current interpreter.
